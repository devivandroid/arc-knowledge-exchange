# Benchmark Notes

This package is designed for evaluating financial reputation and risk models for autonomous agent marketplaces.

## Suggested tasks

- Predict `risk_tier` from transaction behavior.
- Regress `financial_risk_score` using payment history fields.
- Compare rule-based scoring against model-based scoring.
- Detect high-risk synthetic agents using dispute and refund signals.
- Analyze whether active days and counterparty diversity reduce risk.

## Caveats

- The dataset is synthetic and compact.
- It is not an underwriting dataset.
- It does not contain real users, wallets, companies, or marketplace activity.
- It is best used for model prototyping, test pipelines, and governance simulations.
